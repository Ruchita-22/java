
   Node JS
   --------

   Node js  opensource server side technology which is build on 
   chrome v8 javascript Engine.

   it is non-blocking asychronouos(I/O) and cross platform
   which provide environment for developing high scalable and 
   server-side applications.

   Nodejs mainly used to develop  chat appliactions,webapplicationr
   and network application ...etc

    
   node js written  and developed by Ryan Dahl  2009

   javascript  code executes outside the browser.

   nodejs REPL(nodejs shell)
   --------------------------

   Read : it will read input.
   
   Evaluate : it will take input evaluate based on request.

   print : after evaluation the output is displayed

   Loop :  it will loop all the nodejs statements untill close
           the nodejs shell

  .load -- to load the file in to nodejs shell
